# Instagram-Influencer-Analysis
Exploratory Data Analysis on Instagram Influencers dataset using Python
## 📊 Analysis Performed

- Created a bar chart to identify the Top 5 influencers based on followers  
- Used a histogram to analyze the distribution of followers  
- Plotted a scatter plot to study the relationship between followers and influence score  
- Generated a bar chart to visualize country-wise distribution of influencers  

## 📈 Insights
- Top influencers have significantly higher follower counts  
- Most influencers fall within a specific follower range  
- There is a relationship between followers and influence score  
- Certain countries have a higher number of influencers

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
The dataset contains information about Instagram influencers including followers, engagement, and other metrics.

This analysis provides insights into influencer performance and engagement patterns.
